== Description ==

**Custom Cursor Plugin** allows you to replace the default browser mouse cursor with a custom PNG or SVG image on the frontend of your website.

The plugin is lightweight, easy to configure, and uses WordPressâ€™s native Media Uploader for selecting cursor images. It is ideal for creative websites, portfolios, landing pages, and modern UI designs.

The custom cursor follows the mouse smoothly using optimized JavaScript for best performance.

== Frequently Asked Questions ==

= Which image formats are supported? =
PNG and SVG images are supported. Transparent backgrounds are recommended.

= Does this plugin work on mobile devices? =
The plugin is designed for desktop devices. Touch devices do not use a mouse cursor.

= Will this affect the WordPress admin area? =
No. The custom cursor is applied only to the frontend.

= Can I disable the plugin easily? =
Yes. You can disable it using the checkbox in the settings page.

= Is this plugin accessible? =
Custom cursors may impact accessibility. It is recommended to use this plugin only on sites where visual customization is appropriate.
